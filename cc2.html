  工厂函数：
当你想要创建一个对象时可以使用 var obj={} 或 let obj={} 成功创建，但是当你需要创建多个拥有相似属性的对象时，你还会每次都去var？ 这个时候就需要工厂函数，批量生产我们需要的对象，

function cractF(name, age) {
var a = new Object();
a.name = name;
a.age = age;
a.myalert = function() {
alert(this.name + ":"+this.age);
}
return a;
}
var test1 = cractF("江","20");
var test2 = cractF("川","21");
对象确实生产出来了，但是问题又来了，这些对象都是Object生产出来的，可是具体类型是什么呢？于是构造函数出现了；http://www.jb51.net/article/55539.html http://blog.csdn.net/flyingpig2016/article/details/52939679
