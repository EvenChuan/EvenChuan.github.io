Projectajax进一步封装

ajax的进一步精简

发布时间：2017-05-25编辑：EvanChuan阅读（20）

  简单、粗暴、无脑，但是用起来很方便！
  代码：

  function ajax(type, url, str, callback1, callback2) {
    $.ajax({
      type: type,
      url: url,
      data: str || "",
      dataType: "json",
      async: true,
      cache: false,
    success: function (data) {
      callback1 && callback1(data);
  },
    error: function (/*XMLHttpRequest, textStatus, errorThrown*/) {
      /*失败后的返回函数*/
      callback2 && callback2();
  },
    complete: function () {
    /*请求成功请求失败都需要调用的函数，如果有就写在这个位置，没有就不添加*/
    }
   });
 }
  



调用方法：ajax（“post”，“url”，data，fun1，fun2）//即可
