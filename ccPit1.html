针对IE9、IE10 的兼容问题

发布时间：2017-05-25编辑：EvanChuan阅读（20）

  作为前端最讨厌的IE存在的兼容问题
兼容问题描述 存在版本 解决方案
兼容问题描述 存在版本 解决方案
img存在默认的外边框 IE9 border:none
location.origin方法不识别 IE9 为IE9重新加入location.origin方法
图片的绘制上传失效，不支持Canvas IE9 更换其他方法现将图片传入后台，再返回显示
dataset方法失效 IE9 IE10 添加公共方法对元素data-XX属性进行遍历，生成dataset属性
所有的a标签点击时有默认虚线边框 IE9 a:focus{outline:none;}
所有的a标签点击的时候都有个默认的白色背景 IE10 a:active{background:transparent!important}
遇到console代码执行不下去 IE9 打开F2或者将console注释
text-shadow属性不支持 IE9 -webkit-transform:translate(2px, 20px);
-moz-text-shadow:0 0 10px #C06;
-webkit-text-shadow:0 0 10px #c06;
text-shadow:0 0 10px #c06;
filter: Shadow(Color='green', Direction='135', Strength='6')/*Direction阴影的方向*/
filter: dropshadow(OffX=2, OffY=2, Color='black', Positive='true');
border-image属性不支持 IE9 一般很少用
multiple columns不支持 IE9 不用
text-stroke不支持 IE+ 使用filter写出一样的效果
C3动画，渐变 IE9 不支持
resize不支持 IE9 IE10 通过获取宽和高来修改
